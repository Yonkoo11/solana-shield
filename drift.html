Root Cause Analysis

The attack combined social engineering (phishing admin signers via durable nonces) with protocol exploitation(phantom token with fabricated oracle history used as collateral). This was not purely a "human error" - the protocol allowed a new, thinly-traded token to be listed and used as collateral for borrowing real assets.

Vigil cannot prevent phishing. But it detects every on-chain consequence within seconds: the admin privilege transfer, the abnormal withdrawal limit change, and the vault drains. With Guardian's on-chain pause, the team could freeze the protocol before the drain completes.

Prior audits by Trail of Bits (2022) and ClawSecure (February 2026) did not catch this vector. Runtime monitoring catches what static audits miss.
